Who We Are

Computer Vision a track of Westview Robotics that is tasked with detecting specific targets, measuring the distance to them, and sending the information to Systems Control without using human supervision.

What We Do

Students will work with the Linux OS by booting their laptops from a USB with Linux installed (Students without a laptop my request to use a Raspberry Pi). Students use C++ to create Targeting Software that detects targets and measures the distances to them from a camera live feed. To accomplish this, students employ the use of OpenCV, an open source library full of nifty functions that are useful for computer vision. The Targeting Software runs on a Raspberry Pi on the robot, which takes the distance measurements and sends them to Systems Control via networking.

Recommended Prerequisties

FRC Computer Vision is one of the more advanced tracks inside of Option 16 because knowledge of C++ or some other Object Oriented Language (completing AP Computer Science works) is strongly recommended to have. Having a Laptop for every session is strongly recommended, but not required to participate. Previous knowledge of Linux, and individual installation of Linux on studentsâ€™ laptops is not required.

Skills You Will Learn

Students will start by learning the basics of Linux, and how to navigate and use tools on a Linux System. They will also review syntax of C++, and how to compile C++ programs with CMake. Students will later learn other concepts such as networking and collaborative development with GitHub, as well as how basic Vision Processing works using the OpenCV Libraries. Finally, we will install Arch Linux on a Raspberry Pi to go on the Robot, where the final code will run. Offering a very vigorous curriculum, FRC Computer Vision not only offers an introduction to linux and computer vision, but also prepares students for the future by exposing them to a wide array of computer science principles, tools, and problem solving situations.
