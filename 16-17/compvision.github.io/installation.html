Installing Computer Vision Software

Required Packages

In Computer Vision, we use many pieces of software. Here, we've compiled an index of everything you will need

List of Stuff

  • sl (makes you a better typer)
  • git (Amazeballs)
  • lolcat (Extremely Useful, far superior to cat)
  • g++ (How we compile
  • figlet (Fun Thingy)
  • tree (Useful for finding things)
  • open-jdk-8-jre (Java)
  • cmake (How we make make)
