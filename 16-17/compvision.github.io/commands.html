Important Commands

Basic Filesystem Commands

  • ls - lists everything in the current directory
  • cd - change directory
  • rm - remove file
  • mkdir - make directory
  • rmdir - remove directory (only works on empty directories)
  • rm -r - recursively remove, removes directories with stuff in it
  • mv - move/rename file
  • cp - copy file
  • cat - list contents of file

Editing Text

  • nano - opens primitve text editor inside the terminal
  • atom - opens sophisticated text editor outside the terminal, as a GUI
  • vim - opens complex text editor inside the terminal

Fun Commands

  • sl - TRAINS
  • lolcat - cat, but with rainbows
  • figlet - block lettered text
  • cowsay - a cow says stuff

If you can't use them, or they aren't installed, try looking through some FAQs. Email us if you have problems

FAQs

  • The Command Isn't Installed - try installing it via: sudo apt-get install [package name] (ex: sudo apt-get install cowsay)
  • How do I make rainbow text - figlet I Love Rainbows and Unicorns | lolcat
