Github In A Nutshell

Basic Commands

  • git init - create directory
  • git add FILE_NAME - adds a file named "FILE_NAME"
  • git add -A - adds a file for commit
  • git commit -m "MSG" - creates a commit with files you have added
  • git push - Pushes to server (must be preset)
  • git push -u origin master - pushes to remote origin under branch master
  • git remote - shows current remote
  • git remote -v - lists git remotes

If you run into problems, look at the faqs. Email us if you have problems

FAQs

  • "fatal: Not a git repository (or any of the parent directories): .git" - Self Explanatory: you aren't in a git repo. All you have to do is change your directory to the git repo
  • I forgot how to use linux - refer to our fantastic commands reference
